# Mandelbrot Fractal

Программа визуализирующая множество Мандельброта с помощью OpenGL.

## Управление

### Перемещение

w — вверх 

a — влево

s — вниз

d — вправо

### Масштаб

Up — приблизить

Down — отдалить

### Другое

Right — увеличить количество итераций

Left — уменьшить количество итераций

g — увеличить радиус

f — уменьшить радиус
